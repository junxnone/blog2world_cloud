# blog2world_cloud

```
make world cloud with blog summary

optional arguments:
  -h, --help            show this help message and exit
  -t TEXT, --text TEXT  blog summary text
  -f FONT_PATH, --font_path FONT_PATH
                        the path of font
  -sw STOP_WORDS, --stop_words STOP_WORDS
                        the stop words file path
  -o OUTPUT_IMAGE, --output_image OUTPUT_IMAGE
                        the output path
```
